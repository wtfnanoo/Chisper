README

Qu'est-ce que Chisper ?
-----------------------

Chisper est un site internet d'enchères inversées en ligne.
Seules les technologies open source sont utilisées. 

Requirements
------------

Chisper est installé avec Symfony et jQuery. 
Il utilise PHP 5
